What are wire frames?

Wire frames are schematics that you draw before you start coding a project so you can have a visual representation of your website or app to follow as a guideline.

Git Workflow

Git workflow is a feature on github so you can work as a group on projects. instead of having one person work on css html and js you can have 3 people work on the three different codes by utilizing the features workflow. each person will have there own branch and they would push their code to their own branch and the team leader will authorize if the code will be pushed to the master branch.

Ways to Create Responsive Web Design

    Flex Box- adjusts elements to fit certain parameters when the page is changed in resolution
    em- uses percentages and it converts it to pixels
    % Values- instead of pixels using percent values will fit the page to a certain percentage of the screens resolution
    Media Queries- They help keep a div at a certain size
